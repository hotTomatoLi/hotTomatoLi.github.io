为什么函数重载不可以根据返回类型区分

题目

为什么函数重载不可以根据返回类型区分。

解答（知乎）

因为调用时不能指定类型信息，编译器不知道你要调用哪个函数。

当调用max(1, 2);时无法确定调用的是哪个，单从这一点上来说，仅返回值类型不同的重载是不应该允许的。
